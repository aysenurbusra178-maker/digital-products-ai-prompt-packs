# Doctor / Clinic – Prompt Library

## 1. Patient Communication

**Prompt 1 – Appointment Confirmation**
```
Write a clear, friendly appointment confirmation message for a patient.
Date/time: [date time]
Clinic name: [name]
What to bring / prepare: [list]
Tone: warm and professional. Keep under 120 words.
```

**Prompt 2 – Post-Visit Follow-up**
```
Draft a short post-visit follow-up message.
Visit type: [type]
Key instructions given: [list]
When to contact the clinic: [criteria]
Tone: caring and clear. Do not give new medical advice.
```

## 2. Educational Content

**Prompt 3 – Patient Education Snippet**
```
Write a short, easy-to-understand patient education paragraph about [topic].
Audience: general public.
Length: 100–150 words.
Avoid medical jargon or explain it simply.
End with a note to consult their doctor for personal advice.
```

## 3. Reputation & Reviews

**Prompt 4 – Review Response**
```
Write a professional response to a patient review.
Review sentiment: [positive / mixed / negative]
Key points mentioned: [list]
Tone: grateful, professional, and solution-oriented when needed.
```

## 4. Internal Operations

**Prompt 5 – Handover Note**
```
Create a concise internal handover note for clinic staff.
Patient context (non-sensitive summary): [summary]
Pending tasks: [list]
Priority: [high / medium / low]
```
