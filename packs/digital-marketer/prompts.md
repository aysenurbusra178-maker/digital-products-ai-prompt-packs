# Digital Marketer – Prompt Library

## 1. Ad Copy

**Prompt 1 – Meta / Instagram Ad**
```
Write 5 high-converting Meta ad primary texts for:
Offer: [offer]
Audience: [audience]
Goal: [awareness / traffic / conversion]
Tone: [tone]

Include: strong hook, benefit-focused body, clear CTA. Keep under 125 words each.
```

**Prompt 2 – Google Responsive Search Ads**
```
Generate 10 headlines (max 30 characters) and 5 descriptions (max 90 characters) for a Google RSA campaign.
Product/Service: [service]
Primary keyword: [keyword]
Unique value: [USP]
```

## 2. Landing Pages & Funnels

**Prompt 3 – Landing Page Outline**
```
Create a complete landing page outline for [offer].
Sections: Hero (headline + subheadline + CTA), Problem, Solution, Benefits, Social Proof, Offer Details, FAQ, Final CTA.
Write the actual headline and subheadline options (3 each).
```

**Prompt 4 – Email Nurture Sequence**
```
Write a 5-email nurture sequence for leads who downloaded [lead magnet].
Goal: move them to [paid offer].
Tone: helpful, authority-building, never pushy.
```

## 3. Content & SEO

**Prompt 5 – SEO Content Brief**
```
Create a detailed SEO content brief for the keyword “[keyword]”.
Include: search intent, recommended title, H2 structure, word count, internal linking ideas, and key questions to answer.
```

**Prompt 6 – Campaign Creative Brief**
```
Write a creative brief for a [platform] campaign.
Objective: [objective]
Audience: [audience]
Key message: [message]
Must-include elements and restrictions.
```

## 4. Testing & Optimization

**Prompt 7 – A/B Test Hypotheses**
```
Generate 8 strong A/B test hypotheses for improving conversion rate on [page/funnel].
Format: “If we change X to Y, then Z metric will improve because...”
```
