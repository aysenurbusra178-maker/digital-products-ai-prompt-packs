# Content Creator AI Writer – Prompt Library

## 1. Hooks

**Prompt 1 – Scroll-Stopping Hooks**
```
Generate 12 viral hooks for [platform] about [topic].
Audience: [describe]
Tone: [tone]
Make them under 12 words, curiosity-driven, and end with a pattern interrupt or question.
```

**Prompt 2 – Story Hook**
```
Write 5 story-based hooks starting with “I almost…” or “Nobody told me…” for a post about [topic]. Keep authentic and relatable.
```

## 2. Captions

**Prompt 3 – Instagram Caption Formula**
```
Write 5 Instagram captions using this structure:
1. Hook (first line)
2. Value / story (3-5 lines)
3. Soft CTA
4. 5-8 relevant hashtags
Topic: [topic] | Tone: [tone]
```

**Prompt 4 – LinkedIn Thought Leadership**
```
Write a LinkedIn post about [topic] that positions me as an expert.
Structure: Hook → Personal insight → 3 actionable tips → Question for engagement.
Length: 150-220 words. Professional yet conversational.
```

## 3. Scripts

**Prompt 5 – TikTok / Reel Script**
```
Write a 30-45 second TikTok/Reel script on [topic].
Include: visual directions in [brackets], spoken words, on-screen text suggestions, and strong CTA at the end.
```

**Prompt 6 – YouTube Long-form Outline**
```
Create a full YouTube video outline for a 8-12 minute video on [topic].
Sections: Hook (0-15s), Intro, 3 main points with examples, Common mistakes, CTA + end screen.
```

## 4. Ideas & Strategy

**Prompt 7 – 30-Day Content Ideas**
```
Generate 30 content ideas for [niche/platform] based on these pillars: [list 3-4 pillars].
Mix educational, promotional, behind-the-scenes and engagement content.
```

**Prompt 8 – Content Repurposing**
```
I have a [long-form content: blog/YouTube]. Turn it into:
- 5 Instagram carousels
- 3 TikTok scripts
- 1 LinkedIn post
- 5 tweet threads
Keep the core message consistent.
```

## 5. Optimization

**Prompt 9 – SEO Title & Description**
```
Write 5 SEO-optimized titles and meta descriptions for a blog post about [topic]. Primary keyword: [keyword].
```

**Prompt 10 – CTA Optimizer**
```
Rewrite this CTA to increase conversion: “[current CTA]”.
Make it more urgent, specific and benefit-driven for [platform].
```
