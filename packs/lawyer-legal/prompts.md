# Lawyer / Legal – Prompt Library

## 1. Client Intake

**Prompt 1 – Structured Intake Questions**
```
Generate a professional client intake questionnaire for a [practice area] matter.
Include sections for: contact details, opposing party (if any), timeline of events, desired outcome, previous legal steps taken, and budget/timeline expectations.
Keep language clear and non-intimidating.
```

**Prompt 2 – Consultation Summary Email**
```
Write a professional follow-up email after an initial consultation.
Matter type: [type]
Key points discussed: [list]
Next steps agreed: [list]
Tone: clear, reassuring, professional.
```

## 2. Drafting Assistance

**Prompt 3 – Clause Simplification**
```
Rewrite the following contract clause in plain, modern English while preserving legal meaning:
[paste clause]
Also provide a short explanation of what the clause does.
```

**Prompt 4 – Case Status Update**
```
Draft a concise client status update email for a [practice area] matter.
Current status: [status]
Recent developments: [list]
Upcoming milestones: [list]
Keep under 180 words.
```

## 3. Content & Thought Leadership

**Prompt 5 – LinkedIn Legal Insight**
```
Write a LinkedIn post for lawyers about [topic].
Structure: Hook → Practical insight → 2–3 actionable tips → Soft CTA.
Tone: authoritative yet accessible. Avoid legalese where possible.
```

## 4. Difficult Conversations

**Prompt 6 – Scope / Fee Discussion**
```
A client is pushing for work outside the original engagement scope.
Write a polite but firm message that protects the lawyer’s time and clearly explains additional fees or a new engagement letter is required.
```
