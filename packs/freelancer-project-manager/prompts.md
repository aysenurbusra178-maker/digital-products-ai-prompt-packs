# Freelancer Project Manager – Prompt Library

## 1. Client Discovery & Onboarding

**Prompt 1 – Deep Discovery Call Questions**
```
I am about to have a discovery call with a potential client for [project type, e.g. website redesign / brand identity / content strategy].
Client industry: [industry]
My rate range: [your rate]

Generate 12 high-value discovery questions that help me:
- Uncover real goals vs stated goals
- Identify budget reality
- Spot red flags early
- Position myself as the expert
Group them by category (Goals, Constraints, Decision Process, Success Metrics).
```

**Prompt 2 – Client Onboarding Email Sequence**
```
Write a 3-email onboarding sequence for a new freelance client.
Project: [project name]
Client name: [name]
Start date: [date]

Email 1: Welcome + next steps (after contract signed)
Email 2: Kickoff call agenda + materials needed
Email 3: Project communication rules + tools we will use

Tone: professional, clear, confident. Keep each email under 180 words.
```

## 2. Proposals & Scoping

**Prompt 3 – Winning Proposal Outline**
```
Create a complete freelance proposal structure for:
Service: [service]
Client: [client type]
Budget range they mentioned: [amount]
Timeline: [weeks]

Include sections: Understanding, Approach, Deliverables, Timeline, Investment, Next Steps.
Write the “Understanding” section in the client’s language and show that I listened.
```

**Prompt 4 – Scope Creep Defense Script**
```
A client is asking for extra work that is clearly outside the original scope.
Original scope: [list]
Requested addition: [description]

Write a polite but firm message that:
1. Acknowledges the request positively
2. Explains it is out of scope
3. Offers two options (extra fee or move to next phase)
4. Protects the relationship
```

## 3. Project Execution

**Prompt 5 – Weekly Status Update**
```
Write a clear weekly status update email for a client.
Project: [name]
This week’s progress: [bullets]
Blockers: [if any]
Next week plan: [bullets]
Hours used vs budget: [numbers]

Tone: transparent, proactive, no excuses.
```

**Prompt 6 – Risk & Delay Communication**
```
I need to inform the client that [reason for delay] will push the deadline by [X days].
Write a message that owns the situation, offers a recovery plan, and rebuilds confidence.
```

## 4. Invoicing & Payments

**Prompt 7 – Invoice Follow-up Sequence**
```
Create a 3-step polite follow-up sequence for an unpaid invoice.
Invoice amount: [amount]
Due date: [date]
Days overdue: [number]

Step 1: friendly reminder
Step 2: firmer + payment link
Step 3: final notice before stopping work
```

**Prompt 8 – Rate Increase Message**
```
I want to raise my rates for existing clients by [percentage] starting [date].
Write a message that explains the value increase without sounding greedy.
```

## 5. Closing & Case Studies

**Prompt 9 – Case Study Generator**
```
Turn this project into a short case study:
Client: [name/industry]
Challenge: [problem]
My solution: [what I did]
Results: [metrics or qualitative]

Write in first person, 180–220 words, suitable for portfolio or LinkedIn.
```

**Prompt 10 – Testimonial Request**
```
Write a short, warm message asking a happy client for a testimonial.
Make it easy for them (suggest 2–3 specific questions they can answer).
```
