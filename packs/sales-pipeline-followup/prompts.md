# Sales Pipeline + Follow-up Prompt Library

## Lead Qualification
**Prompt 1**
```
I just got a new lead. Industry: [industry]. Budget range mentioned: [amount]. Generate 8 BANT-style qualification questions (Budget, Authority, Need, Timeline) that sound natural in a discovery call.
```

## Proposal Writing
**Prompt 2**
```
Write a concise, high-converting proposal outline for:
Service: [service]
Client problem: [problem]
Proposed solution & timeline: [details]
Investment: [price]
Include: Understanding, Approach, Deliverables, Timeline, Investment, Next Steps.
```

## Follow-up Sequences
**Prompt 3**
```
Create a 4-email follow-up sequence after sending a proposal.
Day 0: Proposal sent confirmation
Day 3: Value reminder + soft ask
Day 7: Social proof + FAQ
Day 12: Final polite close or open loop
Tone: professional, helpful, never pushy.
```

## Objection Handling
**Prompt 4**
```
Client said: “[objection]”. Write 3 different professional responses that acknowledge the concern, reframe value, and move the conversation forward.
```

## Closing
**Prompt 5**
```
Write a short, confident closing message after a positive discovery call. Include clear next steps and a soft deadline for decision.
```
