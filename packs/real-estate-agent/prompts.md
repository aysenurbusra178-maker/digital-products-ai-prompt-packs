# Real Estate Agent – Prompt Library

## 1. Listing Descriptions

**Prompt 1 – Compelling Listing Description**
```
Write a high-converting real estate listing description for:
Property type: [type]
Location: [city/neighborhood]
Key features: [list 5–7 features]
Price: [price]
Target buyer: [first-time / investor / family / luxury]

Tone: warm, aspirational, specific. Include lifestyle benefits, not just square footage. End with a soft call-to-action. Length: 180–250 words.
```

**Prompt 2 – Short Social Listing Teaser**
```
Create 5 short social media teasers (Instagram / Facebook / TikTok) for this listing. Each under 40 words. Include one strong visual suggestion in [brackets].
```

## 2. Client Communication

**Prompt 3 – Buyer Consultation Agenda**
```
Create a professional 30–40 minute buyer consultation agenda. Include discovery questions about must-haves, deal-breakers, timeline, financing status and preferred neighborhoods. Make it conversational.
```

**Prompt 4 – Seller Follow-up Sequence**
```
Write a 4-message follow-up sequence for a seller who listed 14 days ago and has had limited showings.
Day 1: Market update + encouragement
Day 5: Suggested price/strategy adjustment options
Day 10: Open house or fresh photo idea
Day 14: Clear next-step recommendation
```

## 3. Objection Handling & Closing

**Prompt 5 – Price Objection Response**
```
A buyer says the price is too high. Write 3 professional, non-defensive responses that acknowledge the concern, provide market context, and keep the conversation open.
```

**Prompt 6 – Closing Script**
```
Write a short, confident closing conversation script after a successful showing. Help the agent ask for the offer without pressure.
```

## 4. Market & CMA Support

**Prompt 7 – Comparative Market Talking Points**
```
Turn these comparable sales data into clear talking points I can use with a seller:
[paste 3–5 comps with price, size, days on market]
Make it easy to understand and supportive of my recommended list price.
```
